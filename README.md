## Maven

This library is published to the central Maven repository and can be used in your Maven POM file like this:

```
<dependency>
  <groupId>org.hisp.dhis</groupId>
  <artifactId>dhis2-support-commons</artifactId>
  <version>1.0</version>
</dependency>
```
